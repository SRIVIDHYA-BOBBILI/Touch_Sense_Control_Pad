# 🖐️ Touch Sense Control Pad (STM32)

## 📌 Overview
This project demonstrates a **Touch Sense Control Pad** using an STM32 microcontroller. It detects human touch input through capacitive sensing and triggers output in real time.

The project focuses on practical implementation of **embedded systems and human-machine interaction (HMI)**.

---

## 🚀 Features
- Capacitive touch sensing
- Real-time response
- Stable and reliable operation
- Embedded firmware using STM32 HAL
- Hardware-level implementation

---

## 🧠 Working Principle
- Touching the pad changes capacitance
- STM32 detects this variation
- Firmware processes the signal
- Output is triggered (LED ON/OFF)

---

## 🛠️ Hardware Used
- STM32 Microcontroller (STM32F429ZI)
- Touch sensing pad / PCB
- LEDs (output indication)
- Power supply

---

## 💻 Software & Tools
- Embedded C
- STM32CubeMX
- STM32 HAL Drivers
- STM32CubeIDE

---


---

## ▶️ How to Run
1. Open `.ioc` file in STM32CubeMX  
2. Generate code  
3. Open project in STM32CubeIDE  
4. Build and flash to STM32 board  
5. Touch the pad to observe LED response  

---

## 📊 Results
- Accurate touch detection  
- Immediate LED response  
- Stable performance under normal conditions

The project was tested on hardware, where touch input successfully controls LED output.
